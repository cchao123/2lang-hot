# 2lang-hot

English | [中文](./README.md)

A VS Code extension that displays trending content from major Chinese platforms in the sidebar, keeping you updated with hot topics while coding.

## Supported Platforms

- **WeChat Reading** - Popular book recommendations
- **Bilibili** - Trending videos
- **36Kr** - Tech and finance news
- **Weibo** - Hot topics
- **Baidu** - Trending searches
- **Toutiao** - Hot news

## Features

- Tree View sidebar display for each platform
- Preview content directly in VS Code or open in browser
- Refresh data for each platform individually
- Enable/disable platform views as needed

## Installation

Search for `2lang-hot` in the VS Code Extensions Marketplace.

## Configuration

The following settings are available in VS Code:

| Setting | Description | Default |
|---------|-------------|---------|
| `2lang-hot.enableWeixin` | Show WeChat Reading hot list | `true` |
| `2lang-hot.enableBilibili` | Show Bilibili hot list | `true` |
| `2lang-hot.enable36kr` | Show 36Kr hot list | `true` |
| `2lang-hot.enableWeibo` | Show Weibo hot list | `true` |
| `2lang-hot.enableBaidu` | Show Baidu hot list | `true` |
| `2lang-hot.enableToutiao` | Show Toutiao hot list | `true` |

## Usage

1. After installation, click the 2lang icon in the sidebar
2. Expand each platform to view trending content
3. Click an item to view details
4. Click the refresh button to update data

## Commands

- `2lang-hot: Refresh WeChat hot list` - Refresh WeChat Reading data
- `2lang-hot: Refresh Bilibili hot list` - Refresh Bilibili data
- `2lang-hot: Refresh 36Kr hot list` - Refresh 36Kr data
- `2lang-hot: Refresh Weibo hot list` - Refresh Weibo data
- `2lang-hot: Refresh Baidu hot list` - Refresh Baidu data
- `2lang-hot: Refresh Toutiao hot list` - Refresh Toutiao data
- `2lang-hot: Channel settings` - Open channel configuration
- `2lang-hot: Open extension settings` - Open extension settings
- `2lang-hot: View documentation` - View usage guide
